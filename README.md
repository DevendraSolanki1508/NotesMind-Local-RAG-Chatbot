# NotesMind-Local-RAG-Chatbot
NotesMind is a local RAG chatbot for querying documents (PDF, MD, TXT, DOCX) offline. It uses hybrid retrieval (BM25 + vectors), reranking, and semantic chunking for accurate results. Built with Ollama (Mistral 7B), ChromaDB, and LlamaIndex, it supports memory, query rewriting, live ingestion, and evaluation with logging.
